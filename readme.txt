Plugin para criação de colunas em templates que utilizam o framework Bootstrap Twitter.

== Installation ==

e.g.

1. Upload `mw_shortcodes/wp_shortcodes.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
